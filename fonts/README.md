# 免费商用字体

## 字体列表

### 意境命名字体

将字体文件命名为"意境关键词.ttf"即可自动被识别。支持"或"、"与"作为分隔符（如"恋爱或情窦.ttf"）。

| 文件名 | 意境关键词 | 匹配示例 |
|--------|------------|----------|
| `恋爱或情窦.ttf` | 恋爱, 情窦 | "恋爱的滋味", "情窦初开" |
| `悠闲时光.ttf` | 悠闲时光, 悠闲, 时光 | "悠闲的下午", "时光荏苒" |
| `民国风情.ttf` | 民国风情, 民国, 风情 | "民国往事", "老上海风情" |
| `洒脱与不羁.ttf` | 洒脱, 不羁 | "洒脱人生", "放荡不羁" |
| `灵魂感触.ttf` | 灵魂感触, 灵魂, 感触 | "灵魂深处", "内心感触" |
| `老字号.ttf` | 老字号 | "百年老字号", "传统老店" |
| `胸襟或志向.ttf` | 胸襟, 志向 | "胸怀大志", "志向远大" |
| `青春或希望.ttf` | 青春, 希望 | "青春无悔", "充满希望" |
| `鸟语花香.ttf` | 鸟语花香, 鸟语, 花香 | "春天来了", "鸟语花香" |

## 自动匹配规则

1. 系统会自动扫描 `fonts/` 目录下所有 `.ttf` / `.otf` 文件
2. 从文件名中提取意境关键词（用"或"、"与"分隔多个关键词）
3. 分析用户输入文本，匹配关键词
4. 匹配到多个字体时，**随机选择一个**最高得分的字体
5. 无匹配时，自动回退到系统字体

## 使用说明

1. 将字体文件放入 `fonts/` 目录
2. 按意境命名文件（如："温暖或治愈.ttf"）
3. 程序启动时自动扫描加载

如果没有匹配到任何字体，程序会自动回退到系统字体。
