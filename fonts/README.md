## 下载安装步骤

1. 从上方链接下载字体文件
2. 将 `.ttf` (或 `.otf`) 文件重命名为上表对应的文件名
3. 放入当前 `fonts/` 目录
4. 重新启动服务，自动识别

如果字体文件不全，程序会自动回退到系统字体。

## 字体风格对应

| 分类 | 适用场景 |
|------|----------|
| **粗黑** | 口号、标语、标题、力量感文字 |
| **宋体** | 诗文、抒情、文艺、感悟 |
| **手写** | 心情、日记、寄语、祝福 |
| **圆润** | 可爱、萌系、甜美、二次元 |
| **无衬线** | 通用、风景、城市、日常 (默认) |

开启"根据文字意境自动选择字体"后，系统会自动根据你输入的文字关键词选择最匹配的字体！
